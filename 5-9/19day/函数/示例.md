# 示例 {#示例}

* 要求：定义函数odd，输出1-100间的偶数

* step1：设置分割符

```
delimiter $$
```

* step2：创建函数

```
create function odd () returns varchar(300)
begin
declare i int default 1;
declare x varchar(300) default '';
while i <=100 do
if i%2=0 then
set x=concat(x,' ',i);
end if;
set i=i+1;
end while;
return x;
end
$$
```

* step3：还原分割符

```
delimiter ;
```

* step4：调用函数

```
select odd();
```
